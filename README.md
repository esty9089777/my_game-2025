# My Game

My Game is a simple browser-based JavaScript game inspired by the Chrome offline game.
The goal is to control the character and avoid obstacles for as long as possible.

## Getting Started

- Clone or download this repository
- Open `homepage1.html` in a modern web browser
- Start playing the game

## Project Structure

MY_GAME/
├── game.html
├── game.js
├── homepage1.html
├── style.css
└── README.md


## Gameplay

- The player controls a character in the browser
- The objective is to avoid obstacles and survive as long as possible
- The game runs entirely in the browser

## Features

- Simple and intuitive gameplay
- Obstacle avoidance mechanics
- Pure HTML, CSS, and JavaScript (no external libraries)
- Runs locally without installation

## Requirements

- Modern web browser
- JavaScript enabled
